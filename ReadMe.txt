1.	Import the entire folder into the Eclipse
2.	For doing the Menu Driven Action, run the Assignment.java file under model folder.
3.	I have attached the required files which stores the primary data required for the Forming team and calculating the metrics also the Database file to perform the same action.
4.	The Program automatically reads the data and store the data into the hashmap so we can enter any options in the menu.
5.	For creating the fresh record, primary data validation is enabled which restricts the user to enter the invalid data.
6.	Calculate Teammetrics option will not be enabled if the no of teams is not five.
7.	The Menu loop will be running infinitely until the user terminates the program or enter the exit option.
8.	For accessing the GUI, need to run the Main.Java file under the Application folder.
